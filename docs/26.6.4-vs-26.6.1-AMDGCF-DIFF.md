# 26.6.4 vs 26.6.1 AMDGCF static comparison

Date: 2026-07-10

This note records read-only Phase 0 analysis of AMD 26.6.4 WHQL against the earlier 26.6.1
findings. The conclusion is that Boot Camp support was **not** added in 26.6.4.

## Hybrid recipe implication

`original-kernel-hybrid` for 26.6.4 keeps:

- 25.2.1 original `amdkmdag.sys` (`E04E8054…`)
- 26.6.4 original `amdxx64.dll` (`43D9ADA5…`)
- 26.6.4 original `amdgcf.dat` (`E79E42B8…`)

INF-only 26.6.4 remains impossible for the same reasons as 26.6.1. The mandatory path is
hybrid + Microsoft attestation catalog. Hardware E2E validation is tracked in
`docs/HYBRID-E2E-VALIDATION.md`.

## Conclusion

26.6.4 is a version bump only. Boot Camp `020F106B` is still excluded from official INF and GCF
data, and the 26.6.4 kernel still contains the AMDGCF runtime gate. No change to the project
strategy: target `32.0.21043.19003` via hybrid + MS-trusted catalog, not INF-only or local signing.
